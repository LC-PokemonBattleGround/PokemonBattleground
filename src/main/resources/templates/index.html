Bienvenue sur la page d'accueil de notre site Pokémon.

Ce site permet de créer, lister, supprimer des pokémons qui sont stockés en base de données MariaDB en local.


Vous pouvez également faire combattre le pokémon que vous souhaitez face à un pokémon aléatoire de la liste des pokémons éxistant en base de données.


Objectifs :

Ce site web à été développé dans le cadre d'un projet de mise en application d'un site web Spring Boot via l'utilisation de Thymleaf pour les templates.

Ce site est basé sur un cahier des charges que vous pouvez retrouver joint avec le projet Java.
